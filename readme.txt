A WordPress SEO plugin for managing meta tags, generating XML sitemaps, and optimizing content.

== Description ==
SEOPlugin helps optimize your WordPress site for search engines. Features include meta tag management, automatic XML sitemap generation, and a settings page for site-wide SEO configurations.

== Installation ==
1. Upload the `seoplugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure settings under Settings > SEOPlugin Settings.
4. Use the `[seoplugin_keywords]` shortcode to display keyword suggestions.
